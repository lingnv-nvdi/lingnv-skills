# 女帝领导力脚本

本目录包含用于测试和实践女帝领导力技能的辅助脚本。

## 文件说明

### 1. leadership_self_check.py

**功能**：领导力自检问卷

**使用方法**：
```bash
python leadership_self_check.py
```

**内容**：
- 五大领导力要素的自检问题
- 评分系统
- 反馈与建议

### 2. decision_framework.py

**功能**：决策力训练框架

**使用方法**：
```bash
python decision_framework.py
```

**内容**：
- 决策矩阵
- 利弊分析模板
- 决策日志

### 3. crisis_simulation.py

**功能**：危机应对模拟

**使用方法**：
```bash
python crisis_simulation.py
```

**内容**：
- 常见危机场景模拟
- 应对话术练习
- 复盘模板

---

## 使用建议

1. **每日练习**：每天选择一项要素进行反思
2. **每周复盘**：周末回顾一周的领导力表现
3. **刻意改进**：选择一个弱项，集中练习

---

*脚本是辅助工具，真正的领导力在每一次真实互动中修炼。*
