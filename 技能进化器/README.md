# 技能进化器

> 能力的自我迭代系统

## 简介

技能进化器是将"从经验中学习"的过程结构化、可复用化的技能工具。它帮助 AI Agent 将每一次任务执行后的反思转化为可积累的经验，实现技能的精准迭代与进化。

## 核心功能

| 模块 | 功能 | 说明 |
|------|------|------|
| 经验日志 | 记录成功与失败 | 记录任务执行中的关键动作、成功因子、失败教训 |
| 进化日志 | 版本变更追踪 | 记录技能从诞生到成熟的完整演进历程 |
| 版本管理 | 版本号规范 | 主版本.次版本.修订号的三级管理 |
| 补丁更新 | 精准修改 | Find-and-Replace 式的增量更新，而非整体重写 |

## 核心优势

- ✅ **精准迭代**：只改有问题的部分，不破坏原有效逻辑
- ✅ **可追溯**：每次变更都有记录，随时可回滚
- ✅ **轻量化**：Token 消耗低，符合轻量化原则
- ✅ **降低风险**：避免整体重写带来的回归问题

## 设计理念

```
"最好的技能不是一次成型，而是在使用中不断打磨。
  每一次执行后的反思，都是下一次进步的起点。"
```

## 快速开始

### 安装

将本技能文件夹放入您的 skills 目录：

```
skills/
└── 技能进化器/
    ├── SKILL.md           # 主技能文件
    ├── 经验日志/           # 经验记录目录
    ├── 进化日志.md          # 版本变更历史
    ├── 补丁记录/            # 增量补丁文件
    └── references/         # 参考文档
        └── 进化模板.md     # 标准模板
```

### 使用流程

```
┌─────────────────────────────────────────────────────────────┐
│                      技能进化闭环                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   任务执行 ──▶ 结果记录 ──▶ 经验沉淀 ──▶ 优化提炼            │
│                                    │                        │
│                                    ▼                        │
│                               版本更新                       │
│                                    │                        │
│                                    ▼                        │
│                               能力增强                       │
│                                    │                        │
│                                    └──── 回到任务执行         │
└─────────────────────────────────────────────────────────────┘
```

### 触发条件

| 条件 | 阈值 | 动作 |
|------|------|------|
| 同一类任务成功 | 3次 | 提炼为可复用模板 |
| 同一类任务失败 | 1次 | 记录为注意事项 |
| 发现可优化点 | 3个 | 触发补丁更新 |
